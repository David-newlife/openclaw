# docs/tasks

This folder is the default output location for `swarmctl` generated micro-task lists.

- Files like `YYYYMMDD-HHMMSS-tasks.md` are generated artifacts.
- By default these artifacts are ignored by git (see `.gitignore`).

If you need to keep a task list for audit/history, you can:
- copy it elsewhere under `docs/`, or
- temporarily remove the ignore rule and commit it.
