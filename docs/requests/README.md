# docs/requests

This folder is the default output location for `swarmctl` request intake.

- Files like `YYYYMMDD-HHMMSS-request.md` are generated artifacts.
- By default these artifacts are ignored by git (see `.gitignore`).

If you need to keep a request for audit/history, you can:
- copy it elsewhere under `docs/`, or
- temporarily remove the ignore rule and commit it.
